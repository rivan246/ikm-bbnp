== Apliaksi IKM dan Admin ==

Tutorial installasi :
	- setelah ekstrak rar ini, pindahkan folder ke dalam htdocs xampp kalian
	- buka localhost/phpmyadmin (pastikan xampp udah nyala untuk apache dan mysql nya)
	- buat 2 database baru dengan nama 'ikm-bnnp' dan 'admin-ikm'
	- lalu import database dengan file yang ada didalam folder Database ikm
	- kalo udah berhasil, langsung bisa diakses dengan cara akses localhost/ikm

admin : akses dengan localhost/ikm/loginadmin
	user : admin
	pasw : admin123456

bonus dapat diakses lewat 
ikm.vanserver.my.id
